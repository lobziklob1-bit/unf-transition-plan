# План перехода на 1С УНФ

Интерактивный лендинг с управлением задачами, инструкциями и шагами миграции. Данные хранятся в PostgreSQL.

---

## Запуск проекта

Просто напиши мне: **«Запусти проект»** — и я сделаю всё сам.

Или вручную:

### 1. Запустить PostgreSQL
```bash
brew services start postgresql
```

### 2. Запустить сервер
```bash
cd /Users/vladislavlobanov/qwen/unf-transition-plan && node server/index.js
```

### 3. Открыть в браузере
**http://localhost:3001**

---

## Остановка проекта

Напиши мне: **«Останови проект»** или вручную:
```bash
kill $(lsof -i :3001 -t) 2>/dev/null
```

## База данных

- **СУБД:** PostgreSQL 16
- **База:** `unf_transition_plan`
- **Таблицы:** `tasks`, `subtasks`, `workflow_instructions`, `migration_steps`
- **Порт:** `5432`

Все изменения сохраняются в БД:
- ✅ Статусы задач (выполнено / в работе)
- 🔄 Порядок задач (drag-and-drop)
- 📝 Инструкции по взаимодействию
- 📋 Шаги миграции
- ➕ Новые задачи и подзадачи

---

## Что умеет

| Функция | Описание |
|---------|----------|
| **Drag-and-Drop** | Перетаскивание задач за иконку ⋮, порядок сохраняется в БД |
| **Чекбоксы** | Отметка выполнения задач и подзадач |
| **CRUD задач** | Добавление, редактирование, удаление |
| **Инструкции** | Клик по карточке → оверлей с инструкцией, редактирование по паролю |
| **Шаги миграции** | Клик по номеру шага → редактирование (по паролю) |
| **Пароль админа** | `SeregaGornostaevZdoroviyMuzhik` |
